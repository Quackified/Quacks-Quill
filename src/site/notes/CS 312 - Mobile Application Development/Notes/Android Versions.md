#### Cupcake to Kitkat Era (2009 – 2013)

|    Year     | <center>Version</center>     |                Features and Updates                |
| :---------: | :--------------------------- | :------------------------------------------------: |
|   *2009*    | **1.5 - Cupcake**            |                 On-screen keyboard                 |
|   *2009*    | **1.6 - Donut**              |         Support for different screen sizes         |
|   *2009*    | **2.0/2.1 - Eclair**         |     Google maps navigation, multiple accounts      |
|   *2010*    | **2.2 - Froyo**              |      Performance boost, (Adobe) Flash support      |
|   *2010*    | **2.3 - Gingerbread**        | Refined UI, NFC (Near-field Communication) Support |
|   *2011*    | **3.x - Honeycomb**          |                Tablet-only version                 |
|   *2011*    | **4.0 - Ice Cream Sandwich** |             Unified Phone & Tablet UI              |
| *2012-2013* | **4.1 – 4.3 - Jelly Bean**   |      "Project Butter" - Smooth UI, Google Now      |
|   *2013*    | **4.4 - Kitkat**             |           Optimized for low-end devices            |
#### Lollipop to Pie (2014 – 2018)

| API Level |  Year  | <center>Version</center> |               Features and Updates               |
| :-------: | :----: | :----------------------- | :----------------------------------------------: |
|    21     | *2014* | **5.0 - Lollipop**       |            Material Design introduced            |
|    23     | *2015* | **6.0 - Marshmallow**    |       App Permissions, fingerprint unlock        |
|    24     | *2016* | **7.0 - Nougat**         |            Split-screen multi-tasking            |
|    26     | *2017* | **8.0 - Oreo**           | Picture-in-picture (PiP) mode, notification dots |
|    28     | *2018* | **9.0 - Pie**            |      Gesture navigations, Adaptive battery       |
#### Modern Era - Android 10 to Today (2019 – 2025)

| API Level |       Year        | <center>Version</center>   |                                              Features and Updates                                               |
| :-------: | :---------------: | :------------------------- | :-------------------------------------------------------------------------------------------------------------: |
|    29     |      *2019*       | **10 (no codename)**       |                              First version w/o dessert name; System-wide dark mode                              |
|    30     |      *2020*       | **11 (no codename)**       |                                    Chat bubbles , improved privacy controls                                     |
|    31     |      *2021*       | **12 - Snow Cone**         |                                   "Material You" design, more personalization                                   |
|     –     |      *2022*       | **Android 12L**            |                                       Optimized for Foldables and Tablets                                       |
|    33     |      *2022*       | **13 - Tiramisu**          |                                     Enhanced Privacy, multilingual per app                                      |
|    34     |      *2023*       | **14 - Upside-down Cake**  |                                  Security updates, better large-screen support                                  |
|     –     | *2024 (Expected)* | **15 - Vanilla Ice Cream** |                                 Focus on AI Integration, foldable optimization                                  |
|     –     |      *2025*       | **Baklava**                | Live updates notification, Stronger security & privacy, desktop mode improvements, better large-screen support. |
