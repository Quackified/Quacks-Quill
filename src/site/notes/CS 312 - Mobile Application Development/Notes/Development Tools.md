#### Android Studio

**Official IDE** (Integrated Development Environment) for **Android Apps**.

Features the following:
- Code Editor with **syntax highlighting** & **autocompletion**
- **Layout editor** (drag & drop UI design)
- **Emulator for testing** (test app w/out physical phone)
- **Debugging Tools**
- Built-in **Gradle** for app building

#### Java / Kotlin (Programming Languages)

- **Java** - Original Language of Android
- **Kotlin** - Now preferred by Google – concise, safer, interoperable with Java

#### Android SDK (Software Development Kit)

Contains **API libraries**, Tools for compiling, packaging apps, and device emulators.

API (Application Programming Interfaces):
- **Functions** & **Classes** that can be called to **access features** on Android.

	If there was no SDK, you cannot build any Android Apps as there's no official APIs and building tools.

Analogy:
- **Android Studio** - Kitchen
- **Android SDK** - Ingredients

#### Emulator

Simulates different phone models and Android versions for testing without a real device. (Like a Flight Simulator - Pilot)
